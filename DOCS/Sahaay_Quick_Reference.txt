SAHAAY v1.0 — QUICK REFERENCE
================================

PRODUCT
Responsive web care platform for families and elderly loved ones.
The product is Next.js + FastAPI; WhatsApp/Twilio is not the MVP channel.

CORE PROMISE
Within 30 seconds, a family should know:
- Were medicines and meals completed?
- How is the elder feeling?
- Is the routine unusual?
- What is upcoming?
- Is attention needed now?

ROLES
Owner         Creates family, invites members, manages elders and permissions
Family Member Views authorized care data, assigns reminders, receives alerts
Elder         Completes tasks, answers wellness, chats, views calendar, uses SOS

MVP
- Supabase email/password auth, verification, reset, JWT
- Families, invitations, members, RBAC
- Elder profiles, emergency contacts, language/timezone
- Reminder CRUD, recurrence, completion, calendar, escalation
- Daily wellness and activity timeline
- Claude conversation in English/Hindi/Gujarati
- Exa-grounded answers with citations
- Optional ElevenLabs spoken replies
- Deterministic explainable health score
- Family and elder dashboards, analytics, daily summaries
- FCM + in-app notifications, tokens, retries
- Durable idempotent SOS with acknowledge/resolve
- Audit logs and RLS on every public table

STACK
Frontend       Next.js (existing frontend remains unchanged in backend rebuild)
API            FastAPI under /api/v1
Identity       Supabase Auth; FastAPI validates JWT through JWKS
Database       Supabase Postgres with RLS
Jobs           Redis + Celery worker/beat
Push           Firebase Cloud Messaging
AI             Claude + Exa + ElevenLabs

LOCAL COMMANDS (PowerShell)
cd backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
uvicorn app.main:app --reload --port 8000

celery -A app.tasks.celery_app worker --loglevel=info --pool=solo
celery -A app.tasks.celery_app beat --loglevel=info

SAFETY
- Supportive, non-diagnostic AI
- Never change medicine instructions
- Cite current external information
- Explain score components and missing data
- Consent and least privilege for family sharing
- Sahaay does not replace emergency services

NORTH-STAR OUTCOMES
- Elder autonomy and comfort
- Family confidence and faster understanding
- Better reminder/medicine adherence
- Useful, low-noise escalations
- Reliable push/in-app delivery
- Fast SOS acknowledgment
